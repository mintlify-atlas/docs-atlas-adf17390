## What is Life Hunter?

Life Hunter is an intelligent CLI tool that automates opportunity discovery across the web. Configure your profile once with your skills and interests, then let AI-powered agents continuously scrape and match opportunities tailored specifically to you.

## Key Features

<AccordionGroup>
  <Accordion title="🎯 Personalized Profile Matching">
    Configure your skills (TypeScript, React, Python, ML) and interests (AI, startups, web3) once. The AI agents score each opportunity against your profile to surface only what matters to you.
  </Accordion>
  
  <Accordion title="🤖 Parallel Agent Orchestration">
    Four specialized agents scrape different sources simultaneously:
    - **Hacker News Agent** — Top stories + "Who's Hiring" threads
    - **Product Hunt Agent** — Trending products and launches
    - **Reddit Agent** — r/programming, r/webdev, r/MachineLearning, r/forhire
    - **Craigslist Agent** — Software and web development jobs
  </Accordion>
  
  <Accordion title="🧠 AI-Powered Scoring">
    Each scraped item is analyzed by OpenAI GPT-4o-mini, which provides:
    - A relevance score (0-100) based on your profile
    - Type classification (job, deal, or research)
    - A one-sentence AI-generated summary
    Falls back to keyword matching when OpenAI is unavailable.
  </Accordion>
  
  <Accordion title="📊 Beautiful Terminal Interface">
    See your matched opportunities in a clean, colorful terminal table with:
    - Score-based highlighting
    - Source and type indicators
    - Clickable URLs
    - Summary stats (total scraped, matched, run time)
  </Accordion>
</AccordionGroup>

## Quick Example

<Steps>
  <Step title="Install and Configure">
    Install the CLI globally and set up your profile:
    
    ```bash
    npm install -g life-hunter-cli
    life-hunter setup
    ```
    
    Enter your skills, interests, and preferred data sources.
  </Step>
  
  <Step title="Deploy Agents">
    Run the hunt command to scrape all sources:
    
    ```bash
    life-hunter hunt
    ```
    
    Agents scrape in parallel and AI-score each item against your profile.
  </Step>
  
  <Step title="Review Results">
    See a beautiful terminal table with matched opportunities:
    
    ```
    ╔══════════════════════════════════════════════════════════════╗
    ║  Score  │  Type      │  Source          │  Title            ║
    ╠══════════════════════════════════════════════════════════════╣
    ║  95     │  job       │  Hacker News     │  Senior TS Dev    ║
    ║  87     │  research  │  Product Hunt    │  AI Code Tool     ║
    ╚══════════════════════════════════════════════════════════════╝
    ```
  </Step>
  
  <Step title="Get Email Digest">
    Optionally send today's matches as an email:
    
    ```bash
    life-hunter daily
    ```
  </Step>
</Steps>

## How It Works

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────┐
│   CLI User   │────▶│  Convex Backend   │────▶│   OpenAI    │
│  (Terminal)  │◀────│  (Real-time DB)   │◀────│  GPT-4o-mini│
└─────────────┘     └──────────────────┘     └─────────────┘
                            │
              ┌─────────────┼─────────────┐
              ▼             ▼             ▼
         ┌─────────┐  ┌──────────┐  ┌──────────┐
         │   HN    │  │  Reddit  │  │Craigslist│
         │ Agent   │  │  Agent   │  │  Agent   │
         └─────────┘  └──────────┘  └──────────┘
              ▼
         ┌──────────┐
         │  Product │
         │   Hunt   │
         └──────────┘
```

1. **Profile** → Stored in Convex (skills, interests, sources, score threshold)
2. **Hunt** → Agents scrape sources in parallel via Convex actions
3. **Score** → Each item scored by OpenAI (or keyword fallback)
4. **Dedupe** → By URL to avoid duplicates across runs
5. **Display** → Beautiful terminal table with scores & summaries
6. **Daily** → Optional email digest via Nodemailer
